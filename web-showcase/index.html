Robust Claim Predictor

Use Age, BMI, number of Children, and Sex to predict insurance claim amounts with a browser-based model.

About this project

This system implements a robust regression model trained with Huber loss. It predicts insurance claim amounts while reducing the impact of extreme outliers in the training data.

This page is fully virtual: no hardware, backend, or server is required. Predictions happen directly in the browser using pre-computed model weights.

How to use the inputs

Age: customer age in years. Older customers generally have higher claim amounts.

BMI: body mass index. Higher BMI tends to increase insurance risk and claim cost.

Children: number of dependents. More children can raise the expected claim amount.

Sex: categorical predictor encoded as male or female. The model uses sex as a binary feature for insurance risk.

How it works

The browser uses a linear model with coefficients learned from training data. The formula is:

Prediction = intercept + age × weight_age + BMI × weight_bmi + children × weight_children + sex × weight_sex

The training algorithm was Huber regression, which behaves like mean-squared error for normal values and like mean-absolute error for outliers. That makes the model more stable when extreme claim values exist in the data.
